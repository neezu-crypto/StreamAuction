개인정보 처리방침
시행일: 2026년 5월 19일
StreamAuction(이하 "서비스")은 「개인정보 보호법」 등 관련 법령을 준수하며, 이용자의 개인정보를 소중히 보호합니다.
본 처리방침은 서비스가 수집하는 개인정보의 항목·목적·기간 및 이용자 권리를 안내합니다.

제1조 (수집하는 개인정보 항목 및 수집 방법)

수집 경로수집 항목수집 방법
Google 소셜 로그인 Google 계정 이메일, Firebase UID Firebase Authentication — Google OAuth 자동 수집
서비스 이용 중 자동 생성 접속 일시, 잔액(G) 변동 내역, 경매 참여 이력 Firestore · Firebase RTDB 자동 기록
경매 등록 (동의 후 수집) 스트리머 닉네임, SOOP ID 경매 등록 모달 — 이용자 직접 입력 + 개인정보 동의 체크박스 확인 후 저장
보상형 광고 등록 (동의 후 수집) 닉네임, SOOP ID 광고 신청 모달 — 이용자 직접 입력 + 개인정보 동의 체크박스 확인 후 수집
소유자 표시 등록 (동의 후 수집) 닉네임, SOOP ID 매물 상세 페이지 소유자 표시 모달 — 이용자 직접 입력 + 개인정보 동의 체크박스 확인 후 수집. 소유권 이전 시 자동 삭제.

제2조 (개인정보 수집 및 이용 목적)

항목이용 목적
이메일, Firebase UID계정 식별 및 인증, 접속 제한(30인) 게이트, 서비스 내 기능(경매·보상·상점) 제공
접속·이용 이력서비스 운영 통계, 어뷰징 탐지, 잔액 히스토리 관리
닉네임, SOOP ID (경매 등록)경매 매물로 등록 및 서비스 전체 화면에 공개 표시 (검색·랭킹·경매 현황 등)
닉네임, SOOP ID (광고)보상형 광고 슬롯 내 방송국 정보 표시
닉네임, SOOP ID (소유자 표시)매물 상세 페이지 소유자 표시 섹션에 공개 — 방송국 링크(ch.sooplive.co.kr) 제공

제3조 (개인정보 보유 및 이용 기간)

항목보유 기간
Google 계정 정보 (이메일, UID)계정 삭제 요청 시까지 (서비스 종료 시 즉시 파기)
잔액·이력 데이터계정 삭제 요청 시까지
닉네임·SOOP ID (광고)광고 만료 후 운영자 판단 시점에 삭제
닉네임·SOOP ID (소유자 표시)소유권 이전(낙찰·강제청산) 시 자동 삭제. 보유자 직접 삭제 시 즉시 파기.
통계·로그 데이터수집일로부터 1년 (운영 정책에 따라 변경 가능)

제4조 (개인정보의 제3자 제공)

서비스는 원칙적으로 이용자의 개인정보를 외부에 제공하지 않습니다. 단, 다음의 경우는 예외입니다.

예외 사유내용
법령에 의한 경우수사기관 요청 등 법령에 근거한 경우
이용자 동의이용자가 사전에 명시적으로 동의한 경우

제5조 (개인정보 처리 위탁)

수탁사위탁 업무보유 기간
Google LLC (Firebase)인증, 데이터베이스 저장·관리, 서버리스 함수 실행계약 종료 시까지

Firebase 개인정보 처리 방침: firebase.google.com/support/privacy

제6조 (정보주체의 권리·의무 및 행사 방법)

권리행사 방법
열람운영자에게 직접 연락 — 수집된 개인정보 목록 확인 가능
정정·삭제운영자에게 직접 연락 — 계정 삭제 또는 특정 항목 삭제 요청 처리
처리 정지서비스 이용 중단(로그아웃·계정 삭제)으로 처리 정지 효과 발생
동의 철회광고·소유자 표시 등록 시 제공한 닉네임·SOOP ID는 해당 문서 삭제 요청으로 철회 가능

제7조 (개인정보의 안전성 확보 조치)

조치내용
접근 제어Firebase 보안 규칙으로 본인 데이터만 읽기·쓰기 허용, Cloud Functions은 서비스 계정 전용
전송 암호화HTTPS 전 구간 암호화 (Firebase Hosting 기본 적용)
최소 수집닉네임·SOOP ID는 동의 받은 기능(광고·소유자 표시 등)에 한해서만 수집
내부 접근 통제Firebase Console 접근 권한은 운영자 단독 보유

제8조 (개인정보 보호책임자)

책임자오목지면시그 (서비스 운영자)
문의서비스 내 공지 채널 또는 SOOP 채널을 통해 연락

제9조 (닉네임·SOOP ID 동의 체크박스 적용 기준)

원칙: 닉네임 또는 SOOP ID를 직접 입력받아 서비스 화면에 공개적으로 표시하는 모든 폼에는 반드시 개인정보 수집·이용 동의 체크박스를 포함해야 합니다.
동의 체크 전까지는 제출 버튼이 비활성화됩니다.
폼수집 항목표시 위치동의 체크박스

보상형 광고 신청 모달닉네임, SOOP ID광고 슬롯 카드필수
소유자 표시 모달닉네임, SOOP ID매물 상세 페이지필수
[ ] 닉네임 및 SOOP ID가 서비스 화면에 공개적으로 표시되는 것에 동의합니다. (수집 항목: 닉네임·SOOP ID / 목적: 서비스 내 {기능명} 표시 / 보유: 소유권 이전 또는 만료 후 삭제)
